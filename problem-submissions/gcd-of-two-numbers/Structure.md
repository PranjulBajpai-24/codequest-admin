# Problem Structure

Problem Name: "GCD of Two Numbers"
Function Name: gcd
Input Structure: - a: integer - b: integer
Output Structure: - result: integer
Input Fields:
a: First integer
b: Second integer
Output Fields:
result: Greatest common divisor of a and b
Constraints: 1 ≤ a, b ≤ 10^9
Time Limit: 1 second
Memory Limit: 256 MB
